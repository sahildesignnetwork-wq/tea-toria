# 🍵 Tea Toria — Traditional Taste With Tandoor

Tea Toria is Bhopal's authentic tandoor chai café — serving clay-fired kulhad chai, fresh food, and amazing vibes near Kaliasot Dam.

## 🌐 Live Website

**[tea-toria.in](https://tea-toria.in)**

## ✨ Features

- **Home** — Hero section, brand story, brewing process, signature offerings
- **Menu** — Dynamic menu with 11+ categories, powered by admin-uploaded content
- **Services** — Event catering, corporate packages, bulk orders, booking form (WhatsApp integration)
- **About** — Brand story, values, founder section
- **Admin** — Menu Manager dashboard to upload/manage menu items with images & recipes

## 🛠️ Tech Stack

| Technology | Purpose |
|-----------|---------|
| HTML5 | Semantic structure |
| Vanilla CSS | Premium responsive design system |
| Vanilla JS (ES Modules) | Dynamic rendering, WhatsApp integration |
| Vite | Build tool & dev server |
| localStorage | Menu data persistence |


## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
